# Nancy Drew: Тень Восточного Экспресса

Мини-игра в браузере в стиле Nancy Drew:
- сюжет вдохновлён мотивами железнодорожного расследования в духе классического детектива;
- три интерактивные головоломки (шифр, механический замок, логическая дедукция);
- финальная моральная развязка.

> Примечание: в проекте используется авторский пересказ и стилизация, а не дословные цитаты из литературного источника.

## Запуск

Откройте `index.html` в браузере

или через локальный сервер:

```bash
python3 -m http.server 4173
```

После этого перейдите на `http://localhost:4173`.
